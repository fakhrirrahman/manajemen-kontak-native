Tentang Sistem Manajemen Kontak

Aplikasi Web Modern untuk Mengelola Kontak

Sistem Manajemen Kontak adalah solusi lengkap untuk mengorganisir dan mengelola informasi kontak Anda dengan mudah dan efisien. Aplikasi ini dikembangkan menggunakan teknologi web modern untuk memberikan pengalaman pengguna yang optimal.

Fitur Utama

Manajemen Kontak

Tambah, edit, dan hapus kontak dengan informasi lengkap termasuk foto profil, alamat, tanggal lahir, dan catatan khusus.

Filter & Kategori

Organisir kontak berdasarkan kategori (Keluarga, Teman, Kerja, Bisnis) dan prioritas (Tinggi, Sedang, Rendah).

Pencarian Cerdas

Temukan kontak dengan cepat menggunakan fitur pencarian yang dapat mencari berdasarkan nama, email, telepon, atau catatan.

Statistik & Laporan

Lihat statistik kontak, grafik distribusi kategori, dan informasi jumlah kontak yang ditambahkan setiap periode.

Teknologi yang Digunakan

HTML5

Struktur markup semantik untuk tampilan yang terorganisir dan accessible.

CSS3

Styling modern dengan animasi, gradien, dan responsive design.

JavaScript

Interaktivitas dinamis, validasi form, dan pengalaman pengguna yang responsif.

PHP

Server-side processing untuk menyimpan dan mengelola data kontak dalam file text.

Penyimpanan Data

Aplikasi ini menggunakan sistem penyimpanan file text murni untuk menyimpan data kontak, sesuai dengan persyaratan proyek. Data disimpan dalam format berikut:

  • Format JSON: Data kontak disimpan dalam format JSON untuk kemudahan parsing dan pembacaan
  • Format CSV: Backup data dalam format CSV untuk kompatibilitas dengan aplikasi lain
  • File Log: Pencatatan aktivitas dalam file log untuk tracking dan debugging
  • Upload Foto: Foto profil disimpan dalam folder terpisah dengan nama file yang unik

Keamanan & Privasi

Informasi Penting

  • Data kontak disimpan secara lokal di server
  • Validasi input untuk mencegah data yang tidak valid
  • Sanitasi data untuk mencegah injeksi kode berbahaya
  • File upload dengan validasi tipe dan ukuran file
  • Backup otomatis dalam format CSV
